后记

如何看待码云(gitee)大面积故障？ - 知乎

2023年1月2日，我再打开Gitee的时候，这些仓库已经变成空仓库了，一个commit也没有。

这给了我一个启发。Git设计的初衷就是分布式。每个机器都有一份自己的代码仓库。代码托管平台也只是这个网络中的一个公共节点。所以显然不应该把托管平台视为唯一的存储节点。
